WEB SERVICE C# RESTFUL SENDO CONSUMO POR APLICACAO DESKTOP

Exemplo de aplicação desktop feita em C# consumindo Web Services SOAP e RESTful

Conceito utilizando de Desktop X Webservices, outro ponto a verificar que aplicativos mobile de celulares tambem fazem o mesmo consumo
ou seja, com WebServices seja o modo antigo, como o atual feito em WEB API RESTFUL demonstrar a conexao que podemos fazer com qualquer
aplicacao e ou linguagem com uso de WebServices, seja consumido de Mobile, desktop e web

E um exemplo simples, mas funcional para terem uma ideia
